Teddy's Pizzeria

Local Business Website

Project Overview

This is my second website from scratch and my first website designed for a local business; in this case a pizzeria resaurant. My goal was to show I could make a simple yet visually appealing site that an actual business could use and be proud of. I believe I accomplished that goal.

What I Learned

I learned so much in this project. I learned how to be much more efficient typing my code in Atom (especially using the Emmit plugin). I learned to write much cleaner Bootstrap code and incorporate Flexbox into Bootstrap. I learned more about using CSS animations, parallax, and transform effects. I also used the JavaScript library jQuery for the first time to make the lightbox, toggle button, and scroll effects.

Course Landing Page

Long-Form Template

Project Overview

This was the first web page I made from scratch. It is essentially a long-form sales landing page intended to drive conversions for the product (in this case a course). The template is designed to only need to plug in different images, text, and basic styling and be ready to go.

What I Learned

I learned a lot about how to style and layout a web page, especially using Bootstrap. I had trouble at first getting some elements in the desired position, but this was beneficial because it forced me to dive deeper into understanding how Bootstrap works. I also implemented some basic JavaScript to make an accordion style menu. I used stack overflow to help me along the way and then uploaded this my first site to Github. I know it is not the Mona Lisa, but I am proud as it represents not only my beginning but also my persistance and determination.

DevClimb

Web Development Blog

Project Overview

DevClimb is intended to be an ongoing personal project of mine. It is a WordPress blog about web development. The target audience is those looking to learn to code and start their web development career. This site will be my major hub to share all my thoughts and insights, to provide articles and tutorials, and even to offer a coaching service to those at the very beginning of their journey. The futher along in my career I get, the more value I will have to offer, but I know I am always a little ahead of someone else and I would like to help anyone I can.

What I Learned

I have experience making a few WordPress websites in the past; however, after learning to code it has made it even easier to customize and modify the WordPress theme. I gained more experience in WordPress themes and various plugins. I learned more about cPanel and modifying files/settings. I also gained experience in setting up my first email autoresponder and creating lead pages and optin forms. This site is in its infancy stage, but I believe it will end up helping thousands of people in the future.
